Fur 是 .NET 5 平台下极易入门、极速开发的 Web 应用框架。

一分钟入门

.NET 5 平台

.NET 5 是 .NET 的重要且令人兴奋的新方向。你会看到 .NET 变得更加简单，但也有更广泛的功能和实用程序。所有新的开发和功能都将是 .NET 5 的一部分，包括新的 C# 版本。

极易入门

Fur 在设计之初就秉承着 “五极” ：极少入门、极少依赖，极少配置、极其灵活、极易维护 的设计思想，在架构设计和使用上做了大量的优化，支持各个能力阶层的技术人员极速上手。

极速开发

Fur 独具创新的设计理念和汲取了众多优秀大神的框架思想同时结合主流的敏捷开发模式打造的一款极速开发框架。为您节约更多时间，去陪恋人、家人和朋友。
